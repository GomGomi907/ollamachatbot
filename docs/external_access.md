# 외부에서 내 컴퓨터의 챗봇 접속하기

현재 로컬(내 컴퓨터)에서 실행 중인 챗봇 웹사이트(`localhost:3000`)를 외부(스마트폰, 다른 친구의 컴퓨터)에서 접속할 수 있게 만드는 방법입니다.

가장 추천하는 방법은 **ngrok**을 사용하는 것입니다.

---

## 방법 1: ngrok 사용하기 (추천 ⭐)
가장 안정적이고 보안상 안전한 방법입니다.

### 1. ngrok 설치 및 가입
1. [ngrok 공식 홈페이지](https://ngrok.com/)에 접속하여 회원가입을 합니다 (무료).
2. 대시보드에서 **Download**를 클릭하여 윈도우용 ngrok을 다운로드하고 압축을 풉니다.
3. 대시보드에 있는 `Your Authtoken` 명령어를 복사하여 터미널에 입력합니다.
   ```bash
   ngrok config add-authtoken <여기에_토큰_붙여넣기>
   ```

### 2. 터미널 열기
VS Code에서 새 터미널을 엽니다 (`Ctrl` + `Shift` + `\``).

### 3. 터널 실행
**주의**: `ngrok` 명령어가 실행되지 않는다면?
- 다운로드 받은 `ngrok.exe` 파일을 현재 프로젝트 폴더(`c:\Users\tkdrm\Desktop\오토마타 웹사이트`)로 옮겨주세요.
- 그 다음 아래 명령어를 입력합니다 (앞에 `.\`를 붙여야 합니다):
```bash
.\ngrok config add-authtoken <여기에_토큰_붙여넣기>
.\ngrok http 3000
```

만약 그래도 어렵다면 **방법 2: localtunnel**을 사용해보세요 (설치 불필요).

### 4. 접속 주소 확인
터미널에 다음과 같은 화면이 나옵니다:
```
Forwarding                    https://xxxx-xxxx.ngrok-free.app -> http://localhost:3000
```
여기서 `https://xxxx-xxxx.ngrok-free.app` 주소를 복사해서 스마트폰이나 다른 컴퓨터에서 접속하면 됩니다!

---

## 방법 2: localtunnel 사용하기 (간편함 ⚡)
회원가입 없이 바로 사용할 수 있지만, 가끔 연결이 불안정할 수 있습니다.

### 1. 터미널에서 바로 실행
VS Code 터미널에서 다음 명령어를 입력합니다:
```bash
npx localtunnel --port 3000
```

### 2. 접속 주소 확인
```
your url is: https://funny-cat-42.loca.lt
```
위와 같은 주소가 나오면 복사해서 접속합니다.

### 3. 비밀번호 입력 (최초 1회)
localtunnel은 보안을 위해 최초 접속 시 IP 확인을 요청할 수 있습니다. 화면에 나오는 안내에 따라 진행하면 됩니다.

---

## ⚠️ 주의사항
1. **컴퓨터 켜두기**: 내 컴퓨터가 서버 역할을 하므로, 컴퓨터가 켜져 있고 `npm run dev`와 `ollama serve`가 실행 중이어야 합니다.
2. **Ollama 설정**: 챗봇 서버가 내 컴퓨터에서 돌아가므로, Ollama 설정은 바꿀 필요가 없습니다. 외부에서 접속한 사람의 요청도 내 컴퓨터의 Ollama가 처리합니다.
